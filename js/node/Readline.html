package js.node

class js.node.Readline

Available on all platforms

Readline allows reading of a stream (such as process.stdin) on a line-by-line basis.

Note that once you've invoked this module, your node program will not terminate until you've closed the interface.

Static methods

static clearLine (stream:IWritable, dir:ClearLineDirection):Void

Clears current line of given TTY stream in a specified direction.

static clearScreenDown (stream:IWritable):Void

Clears the screen from the current position of the cursor down.

static createInterface (options:ReadlineOptions):Interface

Creates a readline Interface instance. createInterface is commonly used with process.stdin and process.stdout in order to accept user input. Once you have a readline instance, you most commonly listen for the "line" event.

If terminal is true for this instance then the output stream will get the best compatibility if it defines an columns property, and fires a "resize" event on the output if/when the columns ever change (process.stdout does this automatically when it is a TTY).

static cursorTo (stream:IWritable, x:Int, y:Int):Void

Move cursor to the specified position in a given TTY stream.

static moveCursor (stream:IWritable, dx:Int, dy:Int):Void

Move cursor relative to it's current position in a given TTY stream.
