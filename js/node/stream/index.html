package js.node.stream

Duplex

Duplex streams are streams that implement both the Readable and Writable interfaces.
DuplexNewOptions

Options for Duplex private constructor. For stream implementors only, see node.js API documentation
IReadable

IReadable interface is used as "any Readable".
IWritable

Writable interface used for type parameter constraints.
PassThrough

This is a trivial implementation of a Transform stream that simply passes the input bytes across to the output.
Readable

The Readable stream interface is the abstraction for a source of data that you are reading from.
ReadableEvent

Enumeration of events emitted by the Readable class.
ReadableNewOptions

Options for Readable private constructor. For stream implementors only, see node.js API documentation
Transform

Transform streams are Duplex streams where the output is in some way computed from the input.
Writable

The Writable stream interface is an abstraction for a destination that you are writing data to.
WritableEvent

Enumeration for Writable class events.
WritableNewOptions

Options for Writable private constructor. For stream implementors only, see node.js API documentation
