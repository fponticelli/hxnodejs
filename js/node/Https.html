package js.node

class js.node.Https

Available on all platforms

HTTPS is the HTTP protocol over TLS/SSL. In Node this is implemented as a separate module.

Static properties

static globalAgent:Agent

Global instance of Agent for all HTTPS client requests.

Static methods

static createServer (options:TlsServerOptions, ?listener:IncomingMessage -> ServerResponse -> Void):Server

Returns a new HTTPS web server object. The options is similar to Tls.createServer. The requestListener is a function which is automatically added to the 'request' event.

static get (options:HttpsRequestOptions, ?callback:IncomingMessage -> Void):ClientRequest

static get (options:String, ?callback:IncomingMessage -> Void):ClientRequest

Like Http.get but for HTTPS.

static request (options:HttpsRequestOptions, ?callback:IncomingMessage -> Void):ClientRequest

static request (options:String, ?callback:IncomingMessage -> Void):ClientRequest

Makes a request to a secure web server.

options can be an object or a string. If options is a string, it is automatically parsed with Url.parse.

All options from Http.request are valid.
