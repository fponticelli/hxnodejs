package js.node

typedef js.node.InspectOptions

defined in js.node.Util

Available on all platforms

Options object used by Util.inspect.

Properties

showHidden:Null<Bool>

if true then the object's non-enumerable properties will be shown too. Defaults to false.

depth:Null<Int>

Tells Util.inspect how many times to recurse while formatting the object. This is useful for inspecting large complicated objects. Defaults to 2. To make it recurse indefinitely pass null.

customInspect:Null<Bool>

if false, then custom inspect functions defined on the objects being inspected won't be called. Defaults to true.

colors:Null<Bool>

if true, then the output will be styled with ANSI color codes. Defaults to false. Colors are customizable globally via Util.inspect_styles and Util.inspect_colors objects.
